# Pasta de Logos

Esta pasta é destinada para armazenar os logos do projeto.

## Como usar

1. Coloque seus arquivos de logo nesta pasta
2. Para referenciar os logos no código, use o caminho `/logos/nome-do-arquivo.extensao`

## Exemplo de uso

```tsx
// Em um componente React
<img src="/logos/logo.png" alt="Logo da Lanchonete" />

// Ou no HTML
<img src="/logos/logo.png" alt="Logo da Lanchonete" />
```

## Formatos recomendados

- PNG (com transparência)
- SVG (escalável)
- JPG/JPEG (para fotos)

## Tamanhos sugeridos

- Logo principal: 512x512px ou maior
- Favicon: 32x32px ou 64x64px
- Logo para cabeçalho: 200x50px aproximadamente
